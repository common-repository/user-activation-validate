== Description ==
This plugin checks and provides an interface to Admins, giving the option to either delete the user or resend activation link.

== Installation ==
Install
Activate
Go to Users > User Activation Validate

== Frequently Asked Questions ==

= What does it do? =

This plugin helps to minimize spam users in you site by cleaning them in a regular basis based on your own settings.

= User is not showing up in "User Activation Validate" page ? =

When creating user from backend by Administrator, If you don't check the option "Send the new user an email about their account." then it is intended to activate the user by default because of which this created user will be counted as activated.
